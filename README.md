# Power BI Project

A Power BI report developed for the **Databases II** course, analyzing macroeconomic and socioeconomic indicators across multiple dimensions.

---

## 📁 Repository Structure

```
├── Project_latest_version.pbix   # Main Power BI report file
├── data/                         # Source data files (CSV, Excel, etc.)
├── docs/                         # Documentation and report writeups
├── screenshots/                  # Page previews of the report
├── scripts/                      # Any data transformation or SQL scripts
└── README.md
```

---

## Report Pages

The report is organized into the following pages:

| # | Page | Description |
|---|------|-------------|
| 1 | **Our Company** | Overview / introduction page |
| 2 | **Covid Overview** | Impact of COVID-19 on key indicators |
| 3 | **Household** | Household-level economic data |
| 4 | **Tourism Revenue** | Analysis of tourism income trends |
| 5 | **FDI Inflows** | Foreign Direct Investment inflow trends |
| 6 | **Remittance Trends** | Remittance flow analysis |
| 7 | **GDP** | Gross Domestic Product evolution |
| 8 | **Trade_A** | Trade data — Part A |
| 9 | **Trade_B** | Trade data — Part B |
| 10 | **Employment** | Employment rate and workforce indicators |
| 11 | **Inflation** | Inflation trends over time |
| 12 | **Inflation and Consumption** | Relationship between inflation and consumer spending |
| 13 | **Fiscal Response and Debt Trends** | Government fiscal policy response |
| 14 | **Debt** | National/public debt analysis |

---

## Tools & Technologies

- **Power BI Desktop** — Report authoring and data modeling
- **DAX** — Data Analysis Expressions for calculated measures
- **Power Query (M)** — Data transformation and ETL

---

## How to Open

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free).
2. Clone or download this repository.
3. Open `Project_latest_version.pbix` in Power BI Desktop.

---

## Notes

- Data sources may need to be refreshed/relinked after opening on a new machine.
- Place any source data files inside the `data/` folder and update the data source paths in Power BI if needed.
